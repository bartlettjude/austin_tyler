2. **Short bio strip**
   - 2–3 sentences from the existing bio + Austin's portrait + "About me →" (to real /about).
3. **The three collections** (Live / Promotional / Studio)
   - Keep the existing collection copy; each is a tile linking to its gallery, with 3–4 images.
4. **Featured work / named clients** (social proof)
   - Promo covers (Well To Do · Drive Away Her Memory · Wine or Whiskey) + any logos/venues, with permission.
5. **Packages preview**
   - 3–5 package cards w/ price + "Book" CTA → /services.
6. **Testimonials** (once real ones are collected).
7. **Contact / booking CTA band**
   - "Let's work together" + Book button + email + Instagram.
